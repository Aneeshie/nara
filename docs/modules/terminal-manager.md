# Terminal Manager

## Responsibility

Owns every terminal session.

The manager is responsible for creating, destroying and switching between sessions.

---

## Public API

### Create Session

Creates a new terminal session.

---

### Close Session

Terminates the shell and removes the session.

---

### Switch Session

Makes another session active.

---

### List Sessions

Returns every active session.

---
